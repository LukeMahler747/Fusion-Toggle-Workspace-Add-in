# Toggle Workspace — MahlerWorks
Visit us at MahlerWorks.com

Hotkey toggle between the Design and Manufacture workspaces in Autodesk Fusion.

---

## Where to find it in Fusion

Once loaded, the button appears in the Design workspace under:

**Utilities → Utility**

---

## Adding a keyboard shortcut

1. Navigate to **Design → Utilities → Utility** and locate the Toggle Workspace add in in the drop-down menu.
2. Right-click the three dots (elipses) and select **Create Keyboard Shortcut**.
3. Assign any key combination you want (e.g. `Ctrl+Shift+Q`).

The shortcut will then toggle between the Design and Manufacture workspaces from anywhere in Fusion.

---

## Installation

1. Download and unzip the add-in folder.
2. Place the `ToggleWorkspace` folder in your Fusion Add-Ins directory: (On Windows)
   ```
   C:\Users\<YourUsername>\AppData\Roaming\Autodesk\Autodesk Fusion 360\API\AddIns
   ```
3. Open Fusion and go to **Utilities → Add-Ins** (or press `Shift+S`).
4. Find **ToggleWorkspace** in the list under the **Add-Ins** tab and click **Run**.
   - Enable **Run on Startup** if you want it to load automatically every time Fusion opens.

### If it doesn't appear in the list

- **Restart Fusion** and check the list again.
- If it still doesn't show up, click the **+** (plus) button in the Add-Ins dialog and select **Script or Add-In from Device**, then navigate to the `ToggleWorkspace` folder and select it.
